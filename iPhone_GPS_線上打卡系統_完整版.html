iPhone 使用說明

1. 建議上傳到 GitHub Pages、Netlify、Vercel，取得 HTTPS 網址。
2. iPhone 用 Safari 開啟網址，出現定位權限請選「允許」。
3. 若要像 App 一樣使用：Safari → 分享 → 加入主畫面。
4. 若定位失敗：設定 → 隱私權與安全性 → 定位服務 → Safari 網站 → 允許，並開啟「精確位置」。
5. 此測試版資料存在手機瀏覽器 LocalStorage，正式多人共用需改成後端資料庫版。
